### PM 10

Helper to show the name of the 10th Prime Minister of Malaysia

### Installation

`yarn add pm10` or `npm i pm10`

### Usage

```
import { pm10 } from 'pm10';

console.info(pm10());
```

### Options

`prefix: (Boolean)`

Option to show/hide the title prefix, defaults to `false`

```
import { pm10 } from 'pm10';

console.info(pm10({ prefix: true }));
```
